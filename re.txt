Гайд по установке на телефоне

Шаг 1: Подготовка фундамента (Приложения)
Установи три приложения именно из F-Droid (не из Play Store!):

Termux (основной терминал).

Termux:API (мостик между кодом и функциями Android).

Termux:Boot (чтобы сервер включался сам при перезагрузке телефона).

Шаг 2: Настройка системы (Копируй и вставляй)
Открой Termux. Вводи команды по одной. Если видишь запрос (Y/n) — всегда пиши y и жми Enter.

Доступ к памяти: termux-setup-storage (Нажми «Разрешить» в выскочившем окне Android).

Обновление системы: pkg update && pkg upgrade -y

Установка инструментов сборки (чтобы не было ошибок Rust/Pydantic): pkg install python clang rust binutils build-essential libffi openssl -y

Шаг 3: Установка Python-библиотек
Эта часть самая долгая. Если текст замрет на фразе Building wheel — не трогай телефон, он просто «варит» код под твой процессор.


Как быстро заменить текст в nano:
Открой файл: nano main.py

Выдели всё и удали:

Нажми Alt + A (установишь метку начала выделения).

Нажми Alt + / (прыгнешь в самый конец файла, всё выделится).

Нажми Ctrl + K (удалит всё выделенное сразу).

Вставь новый код:

Скопируй код из нашего чата.

Сделай долгий тап по экрану Termux и нажми Paste.

Сохрани и выходи: Ctrl + O, Enter, Ctrl + X.
