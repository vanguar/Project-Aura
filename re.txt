Гайд по установке на телефоне

Шаг 1: Подготовка фундамента (Приложения)
Установи три приложения именно из F-Droid (не из Play Store!):

Termux (основной терминал).

Termux:API (мостик между кодом и функциями Android).

Termux:Boot (чтобы сервер включался сам при перезагрузке телефона).

Шаг 2: Настройка системы (Копируй и вставляй)
Открой Termux. Вводи команды по одной. Если видишь запрос (Y/n) — всегда пиши y и жми Enter.

Доступ к памяти: termux-setup-storage (Нажми «Разрешить» в выскочившем окне Android).

Обновление системы: pkg update && pkg upgrade -y

Установка инструментов сборки (чтобы не было ошибок Rust/Pydantic): pkg install python clang rust binutils build-essential libffi openssl -y

Шаг 3: Установка Python-библиотек
Эта часть самая долгая. Если текст замрет на фразе Building wheel — не трогай телефон, он просто «варит» код под твой процессор.

Установка зависимостей: pip install fastapi uvicorn thefuzz transliterate python-multipart

Шаг 4: Перенос файла main.py
Скинь файл main.py на телефон (через Telegram/Viber) и сохрани его в загрузки.

В Termux перенеси его в «мозг» сервера: cp /storage/emulated/0/Download/main.py ~/main.py

Шаг 5: Настройка Автозапуска (Termux:Boot)
Чтобы тебе не пришлось бегать к телефону мамы после каждой перезагрузки:

Создай папку автозапуска: mkdir -p ~/.termux/boot/

Создай скрипт запуска: echo "termux-wake-lock && python ~/main.py" > ~/.termux/boot/start-aura.sh

Дай права на запуск: chmod +x ~/.termux/boot/start-aura.sh

Важно: Найди на телефоне иконку приложения Termux:Boot и нажми на неё один раз (это активирует автозапуск в системе Android).

Шаг 6: Настройка питания (Анти-сон)
Чтобы Android не «убил» сервер через 5 минут:

Зайди в Настройки телефона -> Приложения -> Termux.

Питание/Батарея -> выбери «Без ограничений» или «Не экономить».

Сделай то же самое для Termux:API.

Шаг 7: Запуск и получение IP
Узнай свой IP: Введи в Termux: ifconfig Ищи блок wlan0. Твой адрес будет после слова inet. Запиши его (например, 192.168.0.15).

Запусти сервер: python ~/main.py

Шаг 8: Финальное соединение с сайтом
Зайди на свой сайт «Ауры» (на Vercel).

Нажми на иконку шестеренки (Settings).

Введи туда IP, который ты узнал в Шаге 7.

Проверь работу: Нажми на кнопку YouTube (она должна стать пульсирующей) и громко скажи: «Новости».

Результат: Через секунду на смартфоне должно открыться приложение YouTube с результатами поиска «Новости».

Твой статус сейчас: Поскольку ты на этапе установки библиотек (Шаг 3), просто дождись завершения. Если Termux снова выдаст красную ошибку — пришли её мне. Если всё пройдет гладко — переходи к Шагу 4.
